# MyHub
模拟实现一个ATM + 购物商城程序\n
作业需求：\n
1.额度 15000或自定义\n
2.实现购物商城，买东西加入 购物车，调用信用卡接口结账\n
3.可以提现，手续费5%\n
4.支持多账户登录\n
5.支持账户间转账\n
6.记录每月日常消费流水\n
7.提供还款接口\n
8.ATM记录操作日志\n
9.提供管理接口，包括添加账户、用户额度，冻结账户等。。。\n
10.用户认证用装饰器\n
注意 解耦 紧耦合 --> 松耦合\n
\n
启动入口：\n
bin -> main.py\n
